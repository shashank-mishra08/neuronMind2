# NeuronMind Design System
*Always consult this document before adding or modifying UI components.*

## 1. Core Philosophy
*   **Aesthetic:** Minimal, Editorial, Premium, Executive, Luxury Enterprise SaaS.
*   **Inspiration:** Apple, Stripe, Linear, OpenAI, Anthropic, Palantir.
*   **Avoid:** Generic SaaS templates, blue medical websites, cyberpunk/robot AI graphics, neon, clutter.

## 2. Color Palette
*   **Primary:** Blue (Deep, professional, trustworthy)
*   **Neutrals:** Graphite, White, Light Gray.
*   **Effects:** Soft glassmorphism (subtle background blurs), thin borders.
*   **Rule:** Never use loud, flashy, or glowing colors.

## 3. Typography
*   **Style:** Large, confident, editorial typography. 
*   **Usage:** Heavy whitespace. Let the text breathe. High contrast for readability.

## 4. UI Components
*   **Cards:** Avoid generic feature cards. Use expansive layouts.
*   **Buttons:** Professional, understated. No glowing or neon buttons.
*   **Gradients:** Avoid "startup" gradients. Stick to solid colors or extremely subtle mesh gradients if necessary for depth.

## 5. Motion Language
*   **Purpose:** Animations must explain ideas (Flow, Coordination, Intelligence, Connection), not just exist to impress.
*   **Style:** Subtle, smooth, deliberate. 
*   **Rule:** Avoid unnecessary or decorative animations.

## 6. Layout & Spacing
*   Heavy use of whitespace to convey premium quality.
*   Structured, grid-based layouts that feel like an operating system interface.
